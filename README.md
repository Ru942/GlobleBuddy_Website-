# GlobleBuddy_Website-
🌍 GlobeBuddy – Premium Travel Website (Responsive Landing Page)  GlobeBuddy is a modern, visually immersive travel and tourism landing page designed to inspire users to explore destinations around the world. It delivers a smooth and elegant user experience with a strong focus on UI design, responsiveness, and clean layout structure.
Built using pure HTML5 and CSS3, this project demonstrates real-world front-end development skills including responsive grid systems, modern styling techniques, and user-friendly interface design without relying on frameworks.

🎯 Project Objective

The main objective of GlobeBuddy is to simulate a real-world travel agency landing page while strengthening front-end development skills. It focuses on layout design, responsiveness, and UI consistency—making it a strong portfolio project for beginners and aspiring web developers.
